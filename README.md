# * Tri N Garden *

Created with love by: `Philip Nguyen`

Tri N Garden is a simple garden tracker application that uses a 2d matrix data structure as a virtual garden.

## How to set up app

* `for repo`
* `npm  install`
* `npm build-dev`
* `npm start`
